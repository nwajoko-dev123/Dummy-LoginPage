# Welcome Back — Dummy Login Page

A clean, modern login page UI built for practicing front-end design and form layout. No real authentication — this is a static front-end mockup.

## Features

- Email and password input fields with icon indicators
- Password visibility toggle ("Show" link)
- "Forgot Password?" link
- Primary "Log in" call-to-action button
- Divider with "or" separator
- Social login buttons (Google, Facebook)
- Soft gradient background with blurred color accents
- Card-based layout with rounded corners and subtle shadow

## Tech Stack

- HTML
- CSS

## Disclaimer

This is a UI practice project only. There is no backend, authentication, or data handling — all fields and buttons are non-functional placeholders for layout/design practice.
